HRVMax Terms

Last updated July 18, 2026

Privacy Policy

HRVMax is a personal recovery dashboard prototype for Oura users. It is currently intended for personal/local use and development testing.

Data We Access

If you connect an Oura account, HRVMax may request access to Oura data such as daily sleep, readiness, detailed sleep, HRV, resting heart rate, sleep duration, temperature deviation, respiratory rate, and SpO2, depending on the scopes you approve.

How Data Is Used

Oura data is used to display your personal dashboard, trend views, and recap statistics. HRVMax does not use your Oura data for advertising, selling data, medical diagnosis, or training machine learning models.

Local Development Storage

In the current local development version, OAuth tokens and synced Oura data are stored on the developer's machine under an ignored local folder. Do not use this prototype with other users' data unless the storage and security model has been upgraded.

Sharing

The current deployed website is informational and does not provide public dashboards or friend sharing. Future social features should require explicit user consent before another person can view Oura data.

Contact

For privacy questions, use the contact email listed in the Oura developer application.
